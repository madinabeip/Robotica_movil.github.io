Intentos

En este proyecto se intentaron implementar varios alforitms que resolvieran el problema de la aspiradora:

  • El primer intento fué un bum and go que siguiera paredes pero al entrar en una sala tendía a estar atascado además de que era incapaz de limpiar ciertas áreas
  • El segundo intento fué el bump and go giratorio, cada vez que la aspiradora se chocaba la velocidad de giro decrecía poco a poco. El problema de este algoritmo era que el robot tendía a no salir del mismo área
  • El tercer intento trató de que cada vez que chocara a un lado, girara al contrario y que cada vez que se diera de frente, fuera havcia atrás. El problema de esto fué que hacia atrás no se miraba si detrás había un obstáculo o no
  • El último intento y el más efectivo fué implementar un giro aleatorio cada vez que la aspiradora se chocara
